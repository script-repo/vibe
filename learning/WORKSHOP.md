## ğŸ“š Workshop Structure

### Hour 1: Foundations (60 minutes)

#### Exercise 1.1: Hello Beautiful World (30 min)
**Learn:** Modern CSS with glassmorphism and gradients
**Build:** A stunning welcome card with frosted glass effects

**Key Concepts:**
- `backdrop-filter` for glassmorphism
- `linear-gradient()` for backgrounds
- Modern CSS layout techniques

#### Exercise 1.2: Make It Interactive (30 min)
**Learn:** JavaScript fundamentals and DOM manipulation
**Build:** An interactive click counter with animations

**Key Concepts:**
- Event listeners
- DOM manipulation
- State management
- CSS transitions

---

### Hour 2: Components & Visualization (60 minutes)

#### Exercise 2.1: Build Reusable Components (30 min)
**Learn:** Component-based architecture
**Build:** A toast notification system

**Key Concepts:**
- Creating elements dynamically
- Reusable functions
- Auto-dismiss timers
- Animation keyframes

#### Exercise 2.2: Visual Magic (30 min)
**Learn:** Canvas API and particle systems
**Build:** Interactive particles that respond to mouse movement

**Key Concepts:**
- Canvas drawing API
- Animation loops with `requestAnimationFrame`
- Mouse event tracking
- Particle physics

---

### Hour 3: Architecture & APIs (60 minutes)

#### Exercise 3.1: Plugin System (30 min)
**Learn:** Extensible architecture patterns
**Build:** A simple plugin registry system

**Key Concepts:**
- Plugin architecture
- Class-based programming
- Dynamic initialization
- Registry pattern

#### Exercise 3.2: API Integration (30 min)
**Learn:** Fetching and displaying external data
**Build:** A user directory from a REST API

**Key Concepts:**
- `fetch()` API
- Async/await
- Error handling
- Loading states

---

### Hour 4: Real-time & Final Project (60 minutes)

#### Exercise 4.1: Real-time Updates (30 min)
**Learn:** Simulating live data updates
**Build:** A real-time dashboard with live stats

**Key Concepts:**
- `setInterval()` for periodic updates
- Dynamic value changes
- Trend indicators
- Smooth transitions

#### Exercise 4.2: Your Masterpiece (30 min)
**Learn:** Combining everything you've learned
**Build:** Your own personal dashboard

**Key Concepts:**
- Integration of all techniques
- Custom design
- Personal creativity
- Full-stack thinking

## ğŸ› Troubleshooting

### Code doesn't run
- Check browser console for errors (F12)
- Make sure HTML is well-formed
- Verify all tags are closed

### Preview is blank
- Check if there's a syntax error
- Click "Run Code" manually
- Reset and try again

### Styles not working
- Make sure styles are in `<style>` tags
- Check CSS syntax
- Verify selectors match elements

### JavaScript errors
- Check for missing semicolons
- Verify all variables are declared
- Make sure function names match
